# Ruboty::Deadline

ruboty plugin for 〆

![screenshot](screenshot.png)

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'ruboty-deadline'
```

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install ruboty-deadline

## Usage

    @ruboty 〆 地球滅亡 3日後 # add 〆
    @ruboty 〆                # list 〆
    @ruboty rm 〆 地球滅亡    # remove 〆
